# Revenue, Donation, and Grant Model

## Free Services

- Website education.
- Local resource directory.
- Emergency sheet templates.
- Basic reminder guides.
- Public safety/privacy doctrine.

## Donation-Supported Services

- Starter care packs.
- Foot protection packs.
- Post-hospital support packs.
- Food stability support.
- Volunteer coordination.
- Community pilot events.

## Paid Optional Services

- NAS/vault setup.
- Apple Watch/iPhone reminder setup.
- Family/caregiver workflow setup.
- Local node implementation for community partners.

## Vendor Sponsorships

Allowed: transparent sponsorship of care packs, events, directory categories, and compute credits.

Not allowed: pay-to-rank hidden referrals, data resale, referral steering without disclosure, access to user health records.

## Unit Economics Starting Assumptions

| Item | Range | Notes |
| --- | --- | --- |
| Starter pack | $35-$75 | low-complexity, high donor fit |
| Foot protection pack | $25-$80 | strong mission fit, clinician boundary |
| Post-hospital pack | $30-$90 | higher coordination value |
| Support request handling | $10-$40 internal time | depends on volunteer/staff |
| NAS setup | $150-$500 service fee plus hardware | paid optional, not required |
| Community event | $250-$1500 | host dependent |

## Grant Targets

- Local community foundations.
- Public health grants.
- Aging/senior support grants.
- Food insecurity grants.
- Digital equity grants.
- Rural health grants.
- Diabetes nonprofit partnerships.


## Non-Negotiable Doctrine

LocalDiabetic does not harvest diabetic data. It coordinates practical local help while keeping health records local-first wherever possible. It does not diagnose, prescribe, dose medication, triage emergencies, or replace licensed medical professionals. Every workflow must have explicit consent, human review, service boundaries, and a path to licensed care or emergency services when needed.
