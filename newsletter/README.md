# The DailyLocal — content library

The newsletter of LocalDiabetic. One small, useful, **grounded** diabetic-life win at a time.
Every issue lives here as markdown, version-controlled, so the list is a real archive — not lost in an inbox.

## The voice (the bar)
- **Useful, not noisy.** One real habit or fact per issue. If it doesn't help someone tomorrow morning, cut it.
- **Plain + scannable.** Short lines, white space, no walls of text. Write like a neighbor, not a brochure.
- **Grounded.** No source → no publish. Diabetic-care content aligns to NIDDK / ADA / IWGDF guidance.
- **Honest.** Educate + organize. Never diagnose, never "your sugar is high, do X." Emergencies → 911.
- **On-brand.** Warm, 🐝, "The good stuff, no noise." Close every issue with `🐝 localdiabetic.com`.

## The shape of an issue
```
1. The hook        — one line: the habit or the truth
2. The why         — why it matters (small things become big things fast)
3. The simple steps — a short, do-it-tomorrow list
4. The tie-in      — "LocalDiabetic is built around simple daily wins like this…"
5. The close       — "The good stuff, no noise."  +  🐝 localdiabetic.com
```
